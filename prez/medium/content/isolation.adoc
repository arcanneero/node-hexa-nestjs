== Les workspaces

----
{
  //...
  "workspaces":[
    "app-models",
    "app-business-api",
    "app-business",
    "app-entrypoint"
  ]
  //...
}
----

== Et on assemble !

.package.json (app-server)
[source, json]
----
{
   "name":"app-server",
   "version":"1.0.0",
   "description":"Server app",
   "main":"src/app.ts",
   "scripts":{
      "start":"tsc src/app.ts --esModuleInterop && node src/app.js"
   },
   "dependencies":{
      "express":"^4.17.2",
